Modules

Since last update 3 more modules have been completed! Module 5 "Developing Websites Locally" was the easiest of these 3 as it introduced the concept of using specified Text Editors for web building - software I already had installed in two instances, one editor for video games and another for digital instrument design. In fact I had already become familiar with the latter in developing the earlier iterations of Growings On.

Before I start tooting my own horn too much Modules 6 and 7 were slightly more challenging. The steepest learning curve so far was found in "Deploying Websites", which although a tough challenge has allowed me to share this newsletter as a link, rather than as a technology challenge in and of itself! As a bonus puzzle I set myself the challenge of hosting vols 1 and 2 in an archive for posterity (links to both can be found at the bottom of the page here. While I did have this system successfully working using Vol 2 as a test subject it was very messy. As I have found previously with problems of this nature they often solve themselves if you leave them alone for a while before coming back to them. My folders are now cleaner than my desk could ever hope of being. It's odd that my brain is much more accepting of the concept of tidy when it comes to file management.

(apologies to Katy)

In between scratching my head with neater web deployment strategies I completed another module on CSS topics covering new approaches to links, which can be seen throughout the page in responsitvity and organisation of buttons and links. There was another learning curve found within this module, that I am actually using this exercise as an opportunity to revise. Can you see any issues in particular with the navigation bar? I feel my eye for design has improved as well, althought I encourage you to be the judge of that and let me know if there are any glaring issues with how the website feels.

Community

In one of the projects I did for Module 7 I was given an open-ended challenge with many possible solutions. I used the solution given as inspiration to try some different approaches and come up with my own ideas, however I ran into a small problem with the formatting of my objects. I shared my quandry with a forum of people taking a range of courses from Codecademy and lo and behold as I was typing this someone replied with a solution! They also gave some glowing feedback on the changes I had made to the example solution.
